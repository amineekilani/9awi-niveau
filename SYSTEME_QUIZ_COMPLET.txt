LOGIQUE DE CORRECTION :
-----------------------

Backend (QuizResultatService.submitQuiz) :
1. Recupere toutes les questions du quiz
2. Pour chaque question :
   - Compare reponse utilisateur avec reponse correcte
   - Incremente compteur si correct
3. Calcule score = (correctes / total) × 100
4. Sauvegarde ResultatQuiz en base
5. Retourne resultat avec details

Frontend (QuizViewerComponent) :
1. Collecte reponses dans Map<questionId, reponse>
2. Calcule temps passe (Date.now() - startTime)
3. Envoie au backend
4. Affiche resultat avec animations
