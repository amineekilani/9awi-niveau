# Images de Badges

Ce dossier contient les images des badges de gamification.

## Images Nécessaires

- `quiz.png` - Badge pour quiz réussis
- `perfect.png` - Badge pour score parfait
- `course.png` - Badge pour cours terminés
- `first-quiz.png` - Badge pour premier quiz
- `first-course.png` - Badge pour premier cours
- `xp.png` - Badge pour XP gagnés
- `level.png` - Badge pour niveau atteint
- `course-completion.png` - Badge par défaut pour parcours terminés

## Format Recommandé

- Taille: 64x64 pixels
- Format: PNG avec transparence
- Style: Cohérent avec le design de l'application
