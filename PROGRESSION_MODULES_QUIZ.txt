================================================================================
                PROGRESSION MODULES ET RESULTATS QUIZ
================================================================================

FONCTIONNALITE AJOUTEE :
-------------------------

Dans la page /cours/:id, pour chaque module, affichage de :
✅ Progression des leçons (X / Y leçons terminées)
✅ Barre de progression visuelle
✅ Statut du quiz (Passé ou Pas encore passé)
✅ Meilleur score du quiz
✅ Nombre de tentatives

================================================================================

BACKEND (3 fichiers) :
----------------------

1. ModuleProgressResponse.java (DTO)
   - Informations du module
   - totalLecons, leconsCompletees, progressionLecons
   - hasQuiz, quizPassed, bestScore, totalAttempts

2. ModuleProgressService.java
   - Calcule la progression pour chaque module
   - Récupère les résultats de quiz
   - Retourne tout en un seul appel

3. ModuleProgressController.java
   - GET /api/module-progress/cours/{coursId}
   - Retourne tous les modules avec progression

================================================================================

FRONTEND (2 fichiers) :
-----------------------

1. module-progress.service.ts
   - Service Angular pour appeler l'API

2. cours-detail.ts et cours-detail.html (modifiés)
   - Charge les modules avec progression pour les étudiants
   - Affiche la progression sous chaque module
   - Affiche les résultats de quiz

================================================================================

LEGENDE :
- Barre bleue : Progression des leçons
- ✓ Passé (vert) : Quiz réussi au moins une fois
- Pas encore passé (gris) : Quiz non tenté
- Meilleur score : Meilleur résultat parmi toutes les tentatives

================================================================================

LOGIQUE DE CALCUL :
-------------------

1. PROGRESSION LECONS :
   - Récupère toutes les leçons du module
   - Compte celles qui sont dans lecon_completions
   - Calcule : (complétées / total) × 100

2. RESULTATS QUIZ :
   - Vérifie si le module a un quiz
   - Récupère tous les ResultatQuiz de l'utilisateur
   - Trouve le meilleur score
   - Compte le nombre de tentatives

POUR TESTER :
-------------

1. REDEMARRER LE BACKEND :
   cd backend
   Ctrl+C
   mvnw.cmd spring-boot:run

2. RAFRAICHIR LE FRONTEND :
   Ctrl+F5

3. TEST ETUDIANT :
   a) Se connecter en tant qu'étudiant
   b) S'inscrire à un cours
   c) Ouvrir le cours (/cours/:id)
   d) Voir la progression sous chaque module :
      - "0 / 3" si aucune leçon complétée
      - "Quiz: Pas encore passé" si quiz non tenté
   
   e) Ouvrir un module
   f) Compléter quelques leçons
   g) Retour au cours
   h) Voir "2 / 3" avec barre de progression
   
   i) Passer le quiz
   j) Retour au cours
   k) Voir "✓ Passé - Meilleur: 80%"
